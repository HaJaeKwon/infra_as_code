# infra_as_code

서로 다른 환경(회사, 집, 클라우드)에서도 동일한 infra 구성이 가능하도록 코드로 구현해 공유합니다.

주로 ansible, docker, vagrant, packer, script 등을 사용합니다.

* [Docker Mysql 가상환경 구축](docker/mysql)
* [Docker PostgreSql 가상환경 구축](docker/postgres)
